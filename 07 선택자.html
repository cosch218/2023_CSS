선택 네 아니오

선택자를 이용한 태그 작성

HTML에서 태그 이름을 적으면 관련 태그가 자동 완성되는 것을 emmet이라고 한다.
emmet으로 작성하는 태그는 선택자 형식으로 작성하면 자동 완성이 된다.

p.class-name이라고 작성하면 클래스가 추가됨

div>p 라고 작성하면 div 안에 p태그를 자식으로 자동 생성

  • div.list>ul>li*3 이라고 작성하면 div 안에 ul 안에 li태그 3개 자동 생성
  • alt 키를 누른채로 입력하고 싶은 부분을 다 누르면 동시에 내용 입력 가능
  • 지우는 것도 마찬가지

div>p*2+h4 라고 작성하면 div안에 p 태그 2개와 h4 태그가 자동 생성

>와 *,+ 를 이용하여 여러개의 태그를 한 번에 작성할 수 있다

()도 사용 가능
