크기

크기는 font-size 속성으로 변경할 수 있고,
기본 크기는 p태그의 글자 크기로 동일하다.
h1 태그의 글자는 이미 200%의 크기로 출력이 되고,
동일한 font-size를 넣어주면 다른 글자들도 동일한 크기가 된다.

안에 있는 글자 크기

부모 태그에 em을 사용하고 자식 태그에도 em을 사용했을 경우,
자식 태그는 100% 크기의 기준을 부모의 값으로 삼는다.
Ex : 부모 태그는 3em, 자식태그는 2em일 경우, 화면에 출력된 자식의 크기는 600%

마진값 -auto

margin 속성값 auto

margin 속성값 auto 값을 사용해줄 때는 display가 block속성(div 주로 사용)이여야 하고,
content 크기가 부모보다 작아야 한다.
(부모 크기와 같으면 마진이 들어갈 공간이 없다.)
부모 크기보다 작은 크기의 content에 margin-auto를 사용하면 부모의 크기 기준으로 중앙 정렬된다.

HTML 태그의 전체 크기 구하기

안녕하세요

태그의 크기를 구하는 방법
1. 컨텐츠의 크기 + 패딩의 크기 + 테두리의 크기 + 마진의 크기
2. CSS에서 box-sizing: border-box; 를 사용

box-size border-box 변경

box-size를 border-box로 변경을 하고 작성하면,
height/width의 값이 컨텐츠 + 패딩 + 테두리의 값으로 바뀐다.
크기값이 중요한 경우에는 크기값 계산이 더 편한 border-box로 바꿔서 작성
